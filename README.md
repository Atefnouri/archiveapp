# React Archive Mangement App 

this is an open source project that aim to create a simple archive and folder mangement system



## Deployment

To deploy this project BE on dev enviorment after installing composer 

```bash
  php artisan serve 
```

## Tech Stack

**Client:** React 

**Server:** Laravel 9.60

**DataBase:** MySQL
